# Energy System Optimization

## Overview
This project analyzes residential energy consumption and evaluates the impact of solar energy on reducing grid dependency using solar generation modeling and sensitivity analysis.

## Project Steps
- Load and inspect the dataset
- Filter residential buildings
- Analyze residential energy usage
- Extract hourly information from timestamp
- Model solar generation
- Calculate grid usage
- Visualize hourly energy behavior
- Compute solar contribution ratio
- Estimate energy saved from the grid
- Perform sensitivity analysis for different solar generation levels

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook / VS Code

## Key Results
- Solar contribution ratio: about 44.6%
- Energy saved from grid: about 183,219 kWh
- Increasing solar generation leads to higher energy savings
